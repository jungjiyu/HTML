11. 크기 속성 값
	: 주의 >> 상대적인 값의 경우 부모 요소(혹은 root 요소) 가 절대 단위로 명시되어있어야됨 
		: 그래야 기준을 잡지

	: 단위 ) 단위라고 다 절대적인게 아니다. 상대 단위도 있다
		:상대 단위 << 가급적 rem 사용이 추천됨. em 의 경우 부모가 누구냐 따라 너무 뒤죽박죽 될 수 있
			em : 
				: em 은 element 의 약자
			rem : 루트 요소(<HTML>)의 크기를 기준으로 비율값 설정
				: root + em
				

		: 절대 단위 >> px, pt

	: 백분율값 >> 부모 "element" 를 기준으로 (상대적으로) 계산된다.
